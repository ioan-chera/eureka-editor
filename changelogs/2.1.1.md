# CHANGES IN Eureka 2.1.1

(Since version 2.1.0)

## Improvements

* Allow linedefs to have both "mapped" and "secret" flags, it was missed.

## Bug fixes

* Fixed fatal error happening when loading DEHACKED lumps with unnamed thing entries.
* Hopefully fixed a rare crash happening when dealing with the thing browser category drop-down.
* Fixed failure to escape backslashes in POSIX paths.
* Fixed endianness problems when saving WAD. Thanks to suve for the fix!

## Platform

* Minimum supported macOS version has been increased to 10.15.
* Fixed failure to run on aarch64 systems. Thanks to DrinkyBird for the patch.
* Fixed integer endianness reportedly failing on platforms like aarch64, current PPC64 machines
  running in little-endian and Loongarch. Thanks to crrodriguez for the patch (#283).
